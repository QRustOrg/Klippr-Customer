# Constants

## Proposito

Documenta el futuro lugar para constantes compartidas del proyecto.

## Responsabilidad en Clean Architecture

Pertenecera a `core/util` para agrupar valores transversales que no pertenezcan a una feature especifica.

## Sin implementacion Kotlin

Este archivo no contiene implementacion Kotlin real ni define constantes activas.

## Futuro archivo

Luego podra convertirse en `Constants` para claves, limites, nombres de preferencias o rutas base compartidas.
