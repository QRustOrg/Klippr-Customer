# Type Converters

## Proposito

Documenta los futuros conversores de tipos usados por Room.

## Responsabilidad en Clean Architecture

Pertenecera a la infraestructura compartida de persistencia para convertir tipos de dominio o datos en formatos compatibles con SQLite.

## Sin implementacion Kotlin

Este archivo no contiene implementacion Kotlin real ni conversores activos.

## Futuro archivo

Luego podra convertirse en `TypeConverters` para fechas, enums, UUID, listas y otros tipos persistibles.
