# Result

Documenta una futura representacion comun de estados de operacion.

## Responsabilidad en Clean Architecture

Pertenecera a `core/util` como contrato compartido para comunicar resultados entre capas sin depender de detalles de UI o red.

## Sin implementacion Kotlin

Este archivo no contiene implementacion Kotlin real ni sealed classes.
