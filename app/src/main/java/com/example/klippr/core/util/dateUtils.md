# Date Utils

## Proposito

Documenta utilidades futuras para manejo de fechas.

## Responsabilidad en Clean Architecture

Pertenecera a `core/util` como soporte transversal para formatear, parsear y normalizar fechas usadas por varias features.

## Sin implementacion Kotlin

Este archivo no contiene implementacion Kotlin real ni funciones de fecha.

## Futuro archivo

Luego podra convertirse en `DateUtils` o extensiones relacionadas con fechas, zonas horarias y formatos visibles.
